2018-09-15 Client completely written in Delphi - the latest build in Tokyo 10.2.3 using VCL components. It allows you to connect to any ownCLOUD server and perform basic operations on files and directories in this system. What may be useful are the options of calling it from the command line with parameters, thanks to which we can automate some of the works on the ownCLOUD cloud.
